generate_sales_data(num_orders=10_000):
- connect to database
- read data from tables Customer, SalesOrderHeader, SalesOrderDetail
	- get the ID_of_latest_customer
	- get the ID_of_latest_SalesOrderID
for i in range(num_orders):
	- random if the order is from old or new customer (30% new customer)
		- if old customer:
            - pick random customerid from 1 to ID_of_latest_customer
            - add new SalesOrderID = ID_of_latest_SalesOrderID + 1 into SalesOrder table
            - generate_SalesOrderDetails for this SalesOrderID
        - if new customer:
            - add new CustomerID = ID_of_latest_customer + 1
            - add new SalesOrderID = ID_of_latest_SalesOrderID + 1 into SalesOrder table
            - generate_SalesOrderDetails for this SalesOrderID
